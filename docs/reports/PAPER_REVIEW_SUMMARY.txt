1. 개인 수준 검증 (최초)
   • 기존: 그룹 평균만 보고
   • 본 연구: 개인별 성능 평가
   • 임상 적용 가능성 직접 평가

2. DTW 기반 혁신적 캘리브레이션
   • 발목 12배 개선 (5.9% → 70.6%)
   • 3단계 파이프라인 (sign + DTW + per-subject)
   • R² 양수 전환 (88.2%)

3. 엄격한 방법론
   • Vicon 금본위 기준
   • 다중 지표 (correlation, R², RMSE, ROM)
   • Deming regression (적절한 통계 방법)

4. 체계적 분석
   • 개인별 상세 분석
   • Best/worst performers 식별
   • Grade 분류 체계

주요 약점 ⚠️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 제한된 표본 크기 ⚠️⚠️⚠️
   • N = 17 (통계적 검정력 부족)
   • 건강한 성인만 (병리적 보행 미검증)
   • 필요: N ≥ 50 (healthy 30 + pathological 20)

2. 음수 R² 문제 ⚠️⚠️⚠️
   • Hip: R² = -9.3 (극악)
   • 의미: 절대값 예측 불가
   • 상대적 변화 추적만 가능

3. 부호 반전 미해결 ⚠️⚠️
   • 76.5%가 음수 상관 (sign inversion)
   • 근본 원인 분석 부재
   • 2D→3D 깊이 모호성 문제

4. Bland-Altman 분석 부재 ⚠️⚠️
   • 일치도(agreement) 평가 미흡
   • Systematic bias 불명확
   • Clinical LoA 미제시

5. 3D 변환 방법 불명확 ⚠️⚠️
   • 2D → 3D 변환 과정 상세 기술 부족
   • Depth 정보 처리 방법 불명
   • Sagittal plane 가정의 제한점 미언급

6. 임상 기준 모호 ⚠️
   • |r| ≥ 0.60 = "Good" 근거 불충분
   • MCID (최소 임상적 의미 차이) 미고려
   • RMSE 43.6° (hip)의 임상적 수용성 불명

7. 일반화 제한 ⚠️
   • Sagittal plane만 (frontal/transverse 제외)
   • 무릎 관절 제외
   • 통제된 환경만 (real-world 미검증)

필수 수정 사항 (Major Revision)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Bland-Altman 분석 추가 ✓
   • Mean bias, 95% LoA 계산
   • Proportional bias 검정
   • Clinical LoA와 비교

2. 3D 변환 방법 상세 기술 ✓
   • Depth 처리 방법 명시
   • Sagittal plane 가정 설명
   • Out-of-plane 오차 정량화

3. 임상 기준 연계 ✓
   • MCID 문헌 검토
   • RMSE vs. MCID 비교
   • 수용 가능 범위 정의

4. 한계점 섹션 강화 ✓
   • R² < 0의 임상적 의미
   • 절대값 vs. 상대 변화
   • 적용 가능/불가능 명확히

5. 부호 반전 원인 분석 ✓
   • 체계적 패턴 분석
   • 카메라 각도 영향
   • 해결 방안 제시

6. Multiple comparison correction ✓
   • Primary vs. secondary outcome
   • Bonferroni correction
   • 또는 FDR 적용

✅ Tier 1 - High Confidence
   • 재활 연구 (group-level)
   • 진행 모니터링 (within-subject)
   • 좌우 비대칭 평가

⚠️ Tier 2 - Moderate Confidence
   • 보행 패턴 분류 (ML)
   • Screening tool
   • 홈 모니터링 (추세)

❌ Tier 3 - Not Recommended
   • 절대값 기반 진단
   • 정밀 ROM 측정
   • Legal/forensic 사용

근거:
  • High |correlation| (0.613-0.678) → 파형 유사
  • R² < 0 → 절대값 신뢰 불가
  • RMSE 43.6° >> MCID 5-10° → 정밀도 부족
