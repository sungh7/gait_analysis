## 비디오 처리 문제

### 왜 637개 비디오에서 388개 CSV만?

**가능한 이유**:

1. **뷰 중복**
   - 같은 시퀀스를 여러 각도에서 촬영
   - 하나의 시퀀스 = 최대 4개 비디오 (front, back, left, right)
   - 388 CSV ≈ 388 독립 시퀀스

2. **처리 실패**
   - 일부 비디오는 MediaPipe 검출 실패
   - 품질 낮음, 조명 불량, 프레임 손상 등
   - 자동으로 스킵됨

3. **중복 파일**
   - 같은 영상의 다른 인코딩
   - 백업 파일
   - 테스트 파일

4. **비 gait 비디오**
   - 보정 영상, 테스트 영상 등
   - gait 분석 대상이 아님
